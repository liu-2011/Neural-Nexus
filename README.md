這是一個關於 **下一代移動運算架構** 的概念驗證專案。
本專案採用 **GNU GPLv3** 授權，致力於技術共享與人類福祉。

## 核心技術 (Research Pillars)
* **Neural-Core:** RISC-V 存算一體架構實驗。
* **Neural-Flow:** 基於光流法的幀生成演算法。
* **Silent-Cool:** 固態散熱方案模擬。
* **Neural-Lumen:** 實驗性 AI 光影重建技術。

**Note:** 本專案由 Liu-2011 於中學時期創立，旨在探索科技服務於人的可能性。
